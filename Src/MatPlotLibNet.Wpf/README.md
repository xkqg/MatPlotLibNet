# MatPlotLibNet.Wpf

WPF chart control for MatPlotLibNet. Renders charts natively via SkiaSharp.

## Quick Start

```xml
<wpf:MplChartControl Figure="{Binding MyFigure}" IsInteractive="True" />
```

All 9 interaction modifiers work: pan (drag), zoom (scroll), 3D rotation (right-drag), rectangle zoom (Ctrl+drag), brush select (Shift+drag), span select (Alt+drag), legend toggle, crosshair, hover tooltip.
