# Cookbook

Runnable code examples with rendered output. Every example uses the fluent `Plt.Create()` API and can be copy-pasted into a .NET 10 console app with `dotnet add package MatPlotLibNet` (and `MatPlotLibNet.Skia` for PNG/PDF).

All images on this page are generated by [`Samples/MatPlotLibNet.Samples.Console/Program.cs`](https://github.com/xkqg/MatPlotLibNet/blob/main/Samples/MatPlotLibNet.Samples.Console/Program.cs) and rendered during CI.
