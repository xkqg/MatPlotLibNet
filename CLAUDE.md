# Project rules

## Git — main repo (MatPlotLibNet)

- By default, only stage with `git add`. Do NOT commit, push, or tag on your own initiative.
- Only run `git commit`, `git push`, or `git tag` when the user **explicitly** asks for it in that conversation.
- Never add `Co-Authored-By` lines to commits.

## Git — wiki repo

- Wiki commits and pushes are allowed freely — no need to wait for explicit permission.
