# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 0.3.x   | :white_check_mark: |
| < 0.3   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in MatPlotLibNet, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, please send an email to the maintainer with:

- A description of the vulnerability
- Steps to reproduce the issue
- The potential impact
- Any suggested fixes (optional)

You can expect an initial response within 72 hours. We will work with you to understand the issue and coordinate a fix before any public disclosure.

## Scope

This policy applies to the MatPlotLibNet NuGet packages and the official npm packages (`@matplotlibnet/*`).

## Disclosure Policy

- We will acknowledge receipt of your report promptly
- We will confirm the vulnerability and determine its impact
- We will release a fix as soon as practical
- We will credit you in the release notes (unless you prefer to remain anonymous)
